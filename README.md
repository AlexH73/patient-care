# PatientCare Project

Приложение для управления данными пациентов клиники на базе Spring Boot.

## Основные функции
*   Регистрация новых пациентов.
*   Просмотр списка всех пациентов и детальной информации.
*   Обновление данных существующих пациентов.
*   Мягкое удаление (soft delete).
*   Поиск пациентов по полу, группе крови и возрасту.
*   Статистика по базе пациентов (общее количество, распределение по полу, количество пациентов старше 60 лет).

## Технологический стек
*   Java 21
*   Spring Boot 4.0.2
*   Maven
*   H2 Database (In-memory)
*   Liquibase (для миграций базы данных)
*   Lombok
*   Swagger (OpenAPI) для документации API

## Как запустить проект
1.  Убедитесь, что у вас установлена Java 21.
2.  Склонируйте репозиторий.
3.  Запустите проект с помощью Maven:
    ```bash
    ./mvnw spring-boot:run
    ```

## API Эндпоинты
Все запросы начинаются с `/api/patients`.

*   `GET /api/patients/info` — приветственное сообщение.
*   `GET /api/patients` — получить список всех активных пациентов.
*   `GET /api/patients/{id}` — получить пациента по ID.
*   `POST /api/patients` — создать нового пациента.
*   `PUT /api/patients/{id}` — обновить данные пациента.
*   `DELETE /api/patients/{id}` — удалить пациента (мягкое удаление).
*   `GET /api/patients/search` — поиск с фильтрами (`gender`, `bloodType`, `ageFrom`, `ageTo`).
*   `GET /api/patients/statistics` — статистика клиники.

## Документация API
